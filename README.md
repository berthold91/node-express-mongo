## 2. Development Steps:

**Step1:** Create a Simple ***"Hello World!"*** App

   - create the Folder Structure ***frontend/src/index.html*** and display ***Hello World***
   - make the frontend of your application to an ***npm project***: run `npm init`
   - install the ***live-server*** package on npm to view the live result of your app: run `npm install -D live-server`
   - open ***frontend/package.json*** and add your ***start script***:  `"start": "live-server src --verbose"`
   - run the frontend of your Application : run `npm start`
   - commit your code
