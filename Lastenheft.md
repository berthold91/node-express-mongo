## Introduction

for this project i will build a ***small Ecommerce Website*** using **Javascript(ES6)** for the frontend, **Node.js(Express)** for the Backend and **mongoDB** as a Database. the project can be use to by Small businesses who want to start selling online quickly...the project should be a ***Web Application*** for now but we might need to build a ***mobile application*** to give the customers of our business a better user experience.
